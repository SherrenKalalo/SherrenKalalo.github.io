My Recent Projects

Hotelo Luxury

Hotelo Luxury is a website that has 2 user roles: user and admin. On the user page, the website provides information about Hotelo Luxury and allows users to make reservations. On the admin page, administrators can view the data entered by users and perform various operations, such as creating, reading, updating, and deleting reservation records.

This project was worked on as a team, and I worked on the frontend part. The website was built using JavaScript language with the React Native framework and Bootstrap. The website uses the React Router DOM library to navigate between pages.

Bapontar

Bapontar is a mobile application that can help users find the best destinations in North Sulawesi. Using this app is effortless. Users just need to register an account and can immediately enjoy its features. Users can view a list of destinations and access descriptions and information about the selected destinations.

This project was a team effort. I worked on the pages for Danau Linow, Pulau Lihaga, Pulau Lembeh, and some sections on the Home page. We built the website using JavaScript, React Native, and Bootstrap. We incorporated the React Native Carousel library to create a captivating sliding effect on the homepage.
